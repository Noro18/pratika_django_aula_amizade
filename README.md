# Pratika Django - Aula Amizade

## Oinsa atu kontorbui 

### 1. Fork Repo ida ne'e 

click fork iha repo [Original](https://github.com/pedrix-48/pratika_django_aula_amizade)

    link: [https://github.com/pedrix-48/pratika_django_aula_amizade](https://github.com/pedrix-48/pratika_django_aula_amizade)

### 2. Clone repo ne'ebe fork ona 

```bash
git clone https://github.com/YOUR_USERNAME/pratika_django_aula_amizade.git
```

### 3. Aumenta upstrea ba repo original 

```bash
git remote add upstream https://github.com/pedrix-48/pratika_django_aula_amizade.git
```

antes halo baut ruam ka aumenta changes tenki run 

```bash
git fetch upstream
git merge upstream/main

# run ida ne'e atu update changes husi repo origin original
```

### 4. Halo branch push no halo PR ba original repo 

```bash
git branch feature-name
git commit -m "commir message"
git push origin feature-name
```

### 5. Halo PR ba orignal branch

ba iha github no halo PR foun atu merge ba origin upsream main
